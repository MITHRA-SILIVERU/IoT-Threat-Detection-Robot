### 📌 Project Overview
This project focuses on a security-centric IoT robot designed to detect hidden threats and identify human presence in restricted or dangerous areas. By integrating advanced sensors with IoT capabilities, the robot provides real-time surveillance and data transmission to ensure safety and proactive threat management.

---

### 🚀 Key Features
* **Human Detection:** Uses specialized sensors to identify human presence accurately.
* **Threat Identification:** Capable of detecting concealed threats in various environments.
* **Real-Time Alerts:** Sends instant notifications to the monitoring station when a threat or human is detected.
* **Wireless Control:** Remote operation and data monitoring via IoT protocols.

---

### 🛠️ Hardware Stack
The system is built using the following components:
* **Microcontroller:** (e.g., Arduino Nano / ESP32)
* **Sensors:** * Human Detection (PIR/Thermal)
  * Threat Detection (Metal Detector/Ultrasonic)
* **Communication:** Wi-Fi Module (for IoT connectivity)
* **Actuators:** DC Motors and L298N Motor Driver for robot mobility

### ⚙️ Setup & Installation
1. **Clone the repo:** `git clone https://github.com/MITHRA-SILIVERU/IoT-Threat-Detection-Robot.git`
2. **IDE:** Install Arduino IDE.
3. **Libraries:** Add necessary libraries for the sensors and WiFi module.
4. **Hardware:** Connect the sensors as per the circuit diagram in the `Documentation/` folder.
5. **Upload:** Flash the code to your microcontroller.
